== README ==

To start server:

java Server                 (defaults to port 50898)

-or-

java Server <port>



To start client:

java Client                (defaults to localhost port 50898)

-or-

java Client <hostname>     (defaults to port 50898)

-or-

java Client <hostname> <port>

-or-

java Client 0 0 -b         (bot game on localhost port 50898)
